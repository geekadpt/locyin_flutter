# locyin_flutter

#### 介绍
集地图+导游+UGC(用户生成内容)于一体，每个用户都可以在地图上发布某个地方曾经发生的事情，比如小时候经历的趣事、屋主的人生经历、地标建筑的兴建变迁和衰落，以及相应的图像资料。再通过技术手段找到相互关联的地点、事件、人物，串联成类似导游手册一样的故事，方便感兴趣的游人去游历、探索和挖掘。
